## ğŸ”§ Customization

### Colors
Modify the CSS variables in `index.css` to change the color scheme:

```css
:root {
  --primary: #2563eb;
  --primary-dark: #1d4ed8;
  --background: #f8fafc;
  --surface: #ffffff;
  /* ... other variables */
}
```

### Typography
The app uses 'Inter' as the primary font. To change it, modify:

```css
body {
  font-family: 'Your-Font', 'Segoe UI', 'Roboto', sans-serif;
}
```
